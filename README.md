Лабораторная работа №2: Объектно-ориентированное программирование на Java
Описание

Данная лабораторная работа посвящена изучению и практическому применению основных концепций объектно-ориентированного программирования (ООП) в языке Java. В ходе работы реализована иерархия классов, демонстрирующая механизмы наследования, инкапсуляции, полиморфизма и абстракции.
Основные концепции ООП в проекте

В проекте реализованы следующие принципы:

&nbsp;&nbsp;&nbsp;&nbsp;Абстракция: Использование абстрактных классов для определения общей структуры сущностей без необходимости создания их экземпляров.

&nbsp;&nbsp;&nbsp;&nbsp;Инкапсуляция: Использование модификаторов доступа (private), а также методов доступа (геттеров и сеттеров) для защиты данных объекта.

&nbsp;&nbsp;&nbsp;&nbsp;Наследование: Создание двухуровневой иерархии классов с использованием ключевого слова extends для повторного использования кода.

&nbsp;&nbsp;&nbsp;&nbsp;Полиморфизм: Переопределение методов родительского класса (@Override) и использование перегрузки конструкторов.

Структура проекта

Проект включает в себя иерархию классов в соответствии с заданием:
1. Абстрактный класс (Верхний уровень)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Определяет базовые характеристики для всей группы объектов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Содержит общие поля (минимум 3).

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Содержит абстрактные и обычные методы, описывающие поведение.

2. Наследуемые классы

    Реализуют специфику конкретных объектов.

    Демонстрируют использование ключевого слова super для вызова конструкторов родителя.

3. Дополнительный функционал

    Статические переменные: В наследуемых классах реализованы уникальные переменные.

    Ввод/Вывод: Реализована демонстрация работы классов с выводом информации в консоль.
